# Productivity-Hub
A modular Python CLI application for managing tasks, notes, expenses, passwords, and productivity tools with secure data storage and clean architecture.

✨ Features
✅ Task Manager
📝 Notes Manager
💰 Expense Tracker
🔐 Password Manager (Encrypted Storage)
📂 JSON-based Data Storage
✔️ Input Validation
🧩 Modular Project Structure
🖥️ Easy-to-use CLI Menu

🛠️ Technologies Used
1.Python 3
2.Object-Oriented Programming (OOP)
3.File Handling
4.JSON
5.Cryptography
6.Modular Programming
7.Git & GitHub

📁 Project Structure

│── main.py
